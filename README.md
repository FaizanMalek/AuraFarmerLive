# Aura Farmer (`aurafarmer.live`)

Next.js app for Aura Farmer—a site where people vote on the aura of public figures and browse leaderboards. Production URL: **https://aurafarmer.live** (configure DNS and hosting when you are ready).

## Stack

- [Next.js 16](https://nextjs.org/) (App Router) + React 19  
- TypeScript, Tailwind CSS v4, ESLint  

Suggested next integrations: Supabase (Postgres + Realtime auth), deploy on Vercel.

## Scripts

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

```bash
npm run build
npm run start
```
